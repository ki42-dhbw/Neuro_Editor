Ein neuronales Netz besteht aus Schichten von Neuronen, die durch gewichtete Verbindungen verknüpft sind. Jedes Neuron berechnet eine gewichtete Summe seiner Eingaben und wendet eine Aktivierungsfunktion an.

→ Forward Pass

Berechnung je Neuron
z = Σ(wᵢ · xᵢ) + b
a = f(z)

z = Vorwärtsaktivierung  ·  w = Gewichte  ·  b = Bias  ·  f = Aktivierungsfunktion  ·  a = Ausgabe (Aktivierung)

⚡ Aktivierungsfunktionen

NameFormelBereichTypischer Einsatz
ReLU max(0, z) [0, ∞) Versteckte Schichten (Standard)
Sigmoid 1 / (1 + e⁻ᶻ) (0, 1) Binäre Klassifikation (Ausgabe)
Tanh (eᶻ − e⁻ᶻ) / (eᶻ + e⁻ᶻ) (−1, 1) Versteckte Schichten (zentriert)
Softmax eᶻᵢ / Σeᶻⱼ (0, 1), Σ=1 Mehrklassen-Klassifikation
LeakyReLU z > 0 ? z : 0.01·z (−∞, ∞) Gegen „tote" Neuronen
Linear z (−∞, ∞) Regressionsausgabe

← Backward Pass (Backpropagation)

Lernziel: Fehler minimieren

Backpropagation berechnet den Gradienten der Verlustfunktion bezüglich aller Gewichte über die Kettenregel:

∂L/∂w = ∂L/∂a · ∂a/∂z · ∂z/∂w

Phasen im Einzelschritt-Modus

  1. → Forward Pass — Eingabe durch das Netz propagieren, alle Aktivierungen a berechnen
  2. ← Backward Pass — Fehler rückwärts propagieren, Deltas δ und Gradienten ∇ berechnen
  3. ⟳ Weight Update — Gewichte mit Optimizer-Regel aktualisieren (Δw)

📉 Verlustfunktionen

NameFormelEinsatz
MSE 1/n · Σ(ŷ − y)² Regression
BCE −[y·log(ŷ) + (1−y)·log(1−ŷ)] Binäre Klassifikation
MAE 1/n · Σ|ŷ − y| Robuste Regression

🔧 Optimierer

SGD — Stochastic Gradient Descent
w ← w − η · ∂L/∂w

η = Lernrate. Einfach, aber anfällig für lokale Minima und Oszillation.

Adam — Adaptive Moment Estimation
m ← β₁m + (1−β₁)g  ·  v ← β₂v + (1−β₂)g²
w ← w − η · m̂ / (√v̂ + ε)

β₁≈0.9, β₂≈0.999. Adaptiert die Lernrate je Gewicht — Standardwahl für die meisten Aufgaben.

RMSProp
v ← γv + (1−γ)g²  ·  w ← w − η · g / √(v + ε)

Gut für RNNs und nicht-stationäre Probleme.

💡 Tipps für den Neural Network Builder

Einzelschritt-Modus: Öffne das Training-Fenster → Tab „③ Training" → „🔬 Einzelschritt". Mit → Forward, ← Backward und ⟳ Update jeden Schritt einzeln beobachten.
Gewicht einfrieren: Rechtsklick auf eine Verbindung → „🔒 Auf 1.0 einfrieren". Das Gewicht bleibt beim Training konstant — nützlich um einzelne Pfade zu isolieren.
Vanishing Gradients: Bei tiefen Netzen mit Sigmoid/Tanh können Gradienten sehr klein werden. Verwende ReLU oder LeakyReLU in versteckten Schichten.
