Vergleich: Menschliches Neuron vs. Künstliches Neuron (KI)

🧠 Menschliches Neuron (Biologie)

  • Dendriten: Empfangen Signale
  • Zellkörper (Soma): Verarbeitet Informationen
  • Axon: Leitet elektrische Impulse weiter
  • Synapsen: Verbindung zu anderen Neuronen
  • Signalart: Elektrochemisch
  • Lernen: Synaptische Plastizität

🤖 Künstliches Neuron (KI)

  • Inputs: Eingangsdaten (x₁, x₂, x₃)
  • Gewichte: Wichtigkeit der Eingaben
  • Summation: Gewichtete Berechnung
  • Aktivierungsfunktion: Entscheidung über Output
  • Output: Ergebnis der Berechnung
  • Lernen: Anpassung der Gewichte (Training)

➡️ Klick, Pfeiltasten oder Buttons zum Navigieren

Aufbau eines Künstlichen Neuronalen Netzes

📥 Eingabeschicht (Input Layer)

  • Erste Schicht des Netzwerks
  • Nimmt Rohdaten auf (Bilder, Zahlen, Text)
  • Jedes Neuron = ein Eingabewert

🧠 Hidden Layers

  • Mehrere Schichten künstlicher Neuronen
  • Erkennen Muster und Zusammenhänge
  • Gewichtete Berechnungen + Aktivierungsfunktionen

📤 Output Layer

  • Letzte Schicht des Netzwerks
  • Liefert das finale Ergebnis
  • z. B. Klassifikation oder Vorhersage

⚙️ Lernprozess

  • Training mit Datensätzen
  • Loss Function misst den Fehler
  • Backpropagation passt Gewichte an
  • Optimierung mit Gradient Descent

Python, TensorFlow und Keras

🐍 Python

  • Beliebte Sprache für KI und Machine Learning
  • Einfache Syntax und viele Bibliotheken
  • Steuert das Training neuronaler Netze
import tensorflow as tf

🧠 TensorFlow

  • KI-Framework von Google
  • Berechnet Matrizen und neuronale Netze
  • Nutzt CPU und GPU für schnelles Training
  • Skalierbar für große Modelle

⚙️ Keras

  • Benutzerfreundliche API auf TensorFlow
  • Erstellt Modelle mit wenigen Zeilen Code
  • Ideal für Schule und Einsteiger
from tensorflow import keras

📊 Beispiel Modell

model = keras.Sequential([ keras.layers.Dense(10, activation="relu"), keras.layers.Dense(1) ]) model.compile(optimizer="adam", loss="mse")
